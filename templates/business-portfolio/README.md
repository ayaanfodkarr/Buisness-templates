## Customization

### Colors

Edit the CSS variables in `style.css`:

```css
:root {
    --bg-white: #ffffff;
    --text-black: #000000;
    --text-gray: #666666;
}
```

### Projects

Add your projects in `script.js`:

```javascript
const projectData = {
    1: {
        title: 'Your Project',
        description: 'Project description...',
        tag: 'Web Design'
    }
};
```
